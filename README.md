# Simple_Mac_Changer
 Introducing SMC, a straightforward MAC changer tool for Linux. Simplify MAC address modification on your device with this easy-to-use tool. Enhance your privacy and security effortlessly. 

 
#How To Use
 
 "-i " / "--interface " : to change network interface
 
 "-r " : to set a random MAC address
 
 "-m " / "--mac_address " : to set a specific MAC address
